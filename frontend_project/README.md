##Frontend-Volatile
众包平台前端项目

###项目目录及说明
--project

----node_modules

----public

----src

--------api: 接口js文件

--------assert：资源文件（图片等）

--------components: vue公共组件

--------layouts: 布局组件

--------plugins: 插件，使用vuetify UI组件库

--------router: index.js页面路由

--------views: 页面以全局组件的形式存放

--------App.vue: 页面入口文件

--------main.js: 程序入口文件，加载各种公共组件
